# Machine-Learning-Project
This repository contains three machine learning projects focused on regression and prediction using real-world datasets.

📁 Project 1: Iris Dataset Analysis & Visualization

Dataset: Iris dataset

🎯 Objective

Perform exploratory data analysis (EDA) and visualization to understand feature relationships.

🔹 Tasks Performed

Loaded dataset using pandas

Displayed shape, columns, .info() and .describe()

Created:

Scatter plots (feature relationships)

Histograms (distribution analysis)

Box plots (outlier detection)

🛠 Tools Used

Python

pandas

matplotlib

seaborn
